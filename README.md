
# 🏏 CCMS – Cricket Club Management System

CCMS is a web-based platform designed to simplify and enhance cricket tournament management. It provides organizers, players, and fans with tools for managing teams, scheduling matches, tracking live scores, and analyzing performance.

---

## 🚀 Features

### 🏆 Tournament Management

* Automated progression from league to knockout stages
* Match scheduling and tracking
* Dynamic winner highlights

### 🏏 Live Scoring System

* Ball-by-ball score updates
* Automatic strike rotation and over handling
* Detailed scorecards with runs, wickets, and extras

### 📊 Analytics & MVP

* Real-time stats: batting average, strike rate, economy
* MVP selection based on performance
* Tournament leaderboards

### 🛡️ Teams & Players

* Team roster management
* Player profiles with stats and match history

### ✨ UI/UX

* Glassmorphism design
* Dark/Light mode support
* Fully responsive layout

---

## 🛠️ Tech Stack

* **Backend:** Python (Flask)
* **Database:** MySQL
* **Frontend:** HTML, CSS, JavaScript
* **Design:** Tailwind-inspired styling

## ⚙️ Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/your-username/CCMS.git
cd CCMS
```

2. Create virtual environment:

```bash
python -m venv venv
venv\Scripts\activate   # Windows
```

3. Install dependencies:

```bash
pip install -r requirements.txt
```

4. Setup database:

* Import the SQL file into MySQL
* Update DB credentials in db.py file

5. Run the app:

```bash
python app.py
```
